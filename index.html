PRISM

Proxy Recognition and Inclusive Scoring Method

Evaluating context dependent bias in LLM based resume screening with a focus on proxy cues and framing effects.

Project Overview

Fall 2025 Showcase LLM Bias Evaluation Resume Screening

PRISM investigates how large language models respond to implicit demographic cues during resume screening, and how evaluator framing (strict merit, culture fit, inclusive framing) changes model decisions.

Our framework measures context dependent bias shifts, capturing patterns that static fairness tests often overlook.

Methods

We curated a structured resume dataset embedding proxy features such as neighborhood indicators, volunteer signals, and prestige cues correlated with demographic attributes.

Resumes were evaluated across multiple prompt framings reflecting real evaluator behavior. Fairness metrics quantify score shifts, ranking changes, and disparity movement across frames.

View Full Poster (PDF)

Key Findings

  • Prompt framing significantly alters how models weigh proxy cues.
  • Inclusive framing reduces disparities in some contexts but can shift bias to other score regions.
  • Static fairness audits may hide bias that only emerges under specific evaluator instructions.
  • PRISM can be integrated as a recurring evaluation framework for production hiring workflows.
