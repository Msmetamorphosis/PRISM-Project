PRISM

Proxy Recognition and Inclusive Scoring Method
Evaluating context-dependent bias in LLM based resume screening with a focus on proxy cues and framing effects.

Project Overview

Fall 2025 Showcase LLM Bias Evaluation Resume Screening

PRISM investigates how large language models respond to implicit demographic cues during resume screening, and how evaluator framing (strict merit, culture fit, inclusive framing) changes model decisions.

Our framework measures context-dependent bias shifts, capturing hidden patterns that static fairness tests often overlook.

Methods

We curated a structured resume dataset embedding proxy features such as neighborhood indicators, volunteer signals, extracurricular cues, and activity descriptions correlated with real demographic patterns.

These resumes were scored using multiple evaluator framings (merit focused, culture fit, triage, inclusive). PRISM logs ranking changes, reasoning patterns, and fairness shifts for each condition.

Key Findings

• Prompt framing meaningfully changes how strongly proxy cues influence ranking decisions.

• Inclusive framing reduces disparities for some groups but can shift bias into different regions of the score distribution.

• A model may appear fair under a static evaluation but reveal inequities under realistic evaluator personas.

• PRISM provides a repeatable framework hiring teams can use whenever prompts or LLM versions are updated.
